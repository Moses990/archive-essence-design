## 2. CAD 中心 + 历史记录

**CAD 中心**（`views/CadCenter.tsx`）：
- 顶部工具条：搜索框（图纸名 / 图号 / 项目）+ 版本状态筛选 Popover（`最新 / 待审阅 / 有冲突 / 已归档`）+ 项目筛选。
- 列表改为按项目分组的可折叠 section（`外滩 22 号 · 128 张` 等）。
- 行点击打开 **右侧抽屉预览**（Sheet）：占位 CAD 缩略图（SVG 网格）、元数据、版本时间线、"打开 / 导出 / 复制路径 / 查看历史"。
- 抽屉状态本地 `useState`，无需路由。

**历史记录**（`views/HistoryView.tsx`）：
- 布局改为 **时间线聚合**：按天分组（今天 / 昨天 / 本周 / 更早），每条带图标（新建 / 更新 / 归档 / 删除 / 索引）。
- 左侧筛选栏：项目、操作类型（多选）、人员、时间范围。
- 每条右侧动作：`查看差异 / 回滚到此版本`（触发确认弹层，仅前端提示 "已回滚（演示）"）。
- 顶部一行"审计摘要"：今日 N 次操作、涉及 M 个项目、K 位成员。

## 3. 项目详情 + AI 中心

**项目详情**（`views/ProjectDetail.tsx`）Tab 内容补齐（当前只有概览）：
- **文件**：树形侧栏 + 文件列表（名称 / 类型 / 大小 / 修改时间 / 版本），顶部搜索与"仅显示我修改的"开关。
- **图纸**：网格视图，卡片含 SVG 占位缩略图、版本徽章、状态点、hover 显示动作。
- **材料**：分类 tabs（石材 / 木饰面 / 五金 / 灯具），卡片含图例色块、供应商、单价、库存状态。
- **团队**：成员表格（头像用首字母色块）、角色、最近活跃、权限；底部"邀请成员"按钮（占位）。
- **活动**：复用历史记录时间线组件（按此项目过滤）。

**AI 中心**（`views/AiCenter.tsx`）：
- 顶部 3 张摘要卡：`本月请求数 / 消耗 tokens / 预计费用`，带 sparkline。
- 中部两栏：
  - 左：**用量趋势**（近 30 天堆叠柱状图，按 provider 着色，纯 SVG）。
  - 右：**模型选择**（默认对话 / 搜索 / 摘要三条任务，各自下拉切换模型）。
- 底部 **提示词模板库**：卡片网格（`图纸说明生成 / 材料匹配 / 会议纪要整理 / 施工图审核清单`），每张含描述、变量数量、"使用 / 编辑 / 复制"。

## 4. 全局与设置

**命令面板**（`CommandPalette.tsx`）：
- 新增 **图纸检索** 分组：输入时匹配示例图纸名 / 图号，选中跳转 CAD 中心并预选。
- 新增 **最近历史** 分组：显示最近 5 条打开的项目 / 图纸，`localStorage` 存储（`vault:recent`）。
- 新增 **筛选预设** 保存：在项目页可"保存当前筛选为预设"，命令面板 `预设` 分组列出可一键应用。预设存 `localStorage` (`vault:filter-presets`)。

**设置**（`views/Settings.tsx`）：
- 新增 **外观** 卡片：主题切换 `深色 / 浅色 / 跟随系统` 三态分段控件。真实切换 `<html>` class（默认已 dark，浅色使用现有 `:root` 变量占位——若无浅色 token 则仅切换 class 并保留深色，作为"演示预览"）。
- 新增 **快捷键** 卡片：分组展示（导航 / 项目 / 图纸 / 全局），每条 `kbd` 样式的按键。
- **本地路径管理** 增强：主根目录 + "附加监听路径"列表，支持添加 / 移除 / 暂停监听（前端状态）。

## 技术要点（面向实现）

- 新组件放到 `src/components/vault/`：
  - `Sheet.tsx`（简单右侧抽屉，基于 shadcn `sheet` 或自建定位 + 遮罩）
  - `charts/StackedBars.tsx`、`charts/DonutUsage.tsx`（纯 SVG）
  - `Timeline.tsx`（复用于 History 与 项目详情/活动）
- 数据集中到 `src/components/vault/data/`：
  - `drawings.ts`（CAD 图纸样例 + 版本历史）
  - `history.ts`（审计事件生成器）
  - `materials.ts`、`team.ts`
  - `aiUsage.ts`（30 天用量、模板）
- `localStorage` 键统一前缀 `vault:`。
- 主题切换：`document.documentElement.classList.toggle('light', mode==='light')`；`跟随系统`用 `matchMedia('(prefers-color-scheme: light)')` 订阅。
- 命令面板"最近历史"由各视图打开条目时调用 `pushRecent(item)` 写入。
